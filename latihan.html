Mari Belajar CSS Bersama

Perlu yang kita ketahui CSS adalah singkatan dari "Cascading Style Sheets" (Lembar Gaya yang Bersusun). Ini adalah bahasa pemrograman yang digunakan untuk mengendalikan tampilan dan tata letak elemen-elemen dalam sebuah halaman web. CSS digunakan untuk memisahkan presentasi visual dari struktur konten dalam HTML, yang membuatnya lebih mudah untuk mengubah gaya dan tampilan halaman web tanpa mengubah struktur dasar kontennya.

Komponen CSS :

  1. Properti (Properties): Properti adalah atribut-atribut yang digunakan untuk menggambarkan bagaimana elemen HTML akan ditampilkan. Misalnya, color digunakan untuk mengubah warna teks, font-size mengatur ukuran teks, dan background-color mengatur warna latar belakang elemen.
  2. Nilai (Values): Nilai-nilai diterapkan pada properti untuk menggambarkan cara elemen akan ditampilkan. Misalnya, pada properti color, nilai blue akan mengubah teks menjadi warna biru.
  3. Selektor (Selectors): Selektor digunakan untuk memilih elemen atau sekelompok elemen di halaman web yang akan diberi gaya. Selektor paling umum adalah selektor elemen (misalnya, 'p' untuk semua elemen paragraf) dan selektor kelas (misalnya, .judul untuk semua elemen dengan kelas "judul").
  4. Aturan dalam Kaskade (Cascade Rules): CSS menggunakan aturan kaskade yang mengatur urutan prioritas ketika ada beberapa aturan yang berlaku pada elemen yang sama. Ini memungkinkan Anda mengontrol bagaimana gaya diterapkan dan menghindari konflik.
  5. Media Query: Media query memungkinkan Anda untuk mengubah tampilan halaman web berdasarkan karakteristik perangkat yang digunakan oleh pengguna, seperti ukuran layar atau orientasi (mode potret atau lanskap). Ini membantu dalam pembuatan tampilan yang responsif.
  6. Pseudo-Class dan Pseudo-Element: Ini adalah selektor khusus yang digunakan untuk menggambarkan keadaan atau bagian dari elemen. Misalnya, :hover adalah pseudo-class yang digunakan untuk menggambarkan saat cursor mouse berada di atas elemen.
  7. Box Model: Model kotak adalah konsep penting dalam CSS yang menggambarkan bagaimana setiap elemen di halaman web diatur dalam kotak, termasuk margin, border, padding, dan content.
  8. Inheritance (Pewarisan): Beberapa properti CSS dapat diwariskan dari elemen induk ke elemen anak. Ini memungkinkan Anda mengatur gaya secara efisien.
  9. Fungsi (Functions): CSS mendukung berbagai fungsi yang memungkinkan Anda untuk melakukan operasi matematika, perubahan warna, dan transformasi lainnya pada nilai-nilai CSS.
